Hi！歡迎你踏上這趟學習 docker 的旅程

我是 Justin，我將會在未來的篇章中將我所理解的 docker 一步步教給你，這趟旅程是循序漸進的，會從最基礎的 docker 概念開始說起，之後慢慢觸及到 docker 中的其他豐富主題，這些主題包括 docker-compose、swarm 等等，我會盡量以最容易理解的話來說明當中的觀念，如果當中還是有遇到問題、不懂的地方，也歡迎向我發問哦！

## 誰適合參加這趟旅程

任何人都可以！我會從最基本的觀念開始講起

但如果你曾有開發過任何系統的經驗，那會更好！因為這些經驗會讓你更能體會 docker 所帶來的美好，更容易理解我所舉的範例

如果你已經對 docker 有部分的熟悉度，你可以直接跳到想學的地方開始

## 快速測驗區塊

在之後每篇文章的下方，你都會發現標題是「快速測驗」的段落，在這我會寫上幾個我認為讀完這篇文章後你應該要能回答出的幾個問題，如果

- 你回答的出來：代表你有吸收到觀念，可以繼續往下學習
- 你回答不出來：代表你有些觀念還不是很了解，可以往上找答案或是上網查，先把他理解後再繼續學習

這個學習法叫做「Active Recall」，是一個經過[實證非常有效的學習方式](https://www.amazon.com/Make-Stick-Science-Successful-Learning/dp/0674729013)，關於更詳細的說明與方法，可以參考 [Ali Abdaal 的這個影片](https://www.youtube.com/watch?v=ukLnPbIffxE&t=9s)

那我們就開始吧！
